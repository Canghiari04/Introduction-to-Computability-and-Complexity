In order to prove our results we need a \textbf{computational model} in the form of an abstract machine, relying on two main assumptions:
\begin{enumerate}
    \item It should be much \textbf{simple} as possible.
    \item It must be able to run any other type of machine.
\end{enumerate}
Our reference model will be the \textbf{Turing Machine}, that guarantees both the previous peculiarities.
