Up to now, we have seen two types of complexity classes, namely:
\begin{itemize}
    \renewcommand{\labelitemi}{-}
    \item \textbf{P}, class of easy problems or \textbf{tractable} problems, that can be solved in polynomial time, even though the polynomial number is huge.
    \item \textbf{EXP}, which contains the whole class \textit{P}, but also some problems which cannot be solved in polynomial time, intrinsically requiring exponential time, named \textbf{intractable} problems.
\end{itemize}
It's now time to study the \textbf{border} of these two classes, between tractable and intractable problems. The border is a great area of classes between
\textit{P} and \textit{EXP}, so there isn't only the complexity class \textbf{NP}.
