In theory of computation, a process is taken to be an \textbf{algorithm}, so we replace the term process by the word algorithm. Any algorithm must satisfy always
these constraints:
\begin{enumerate}
    \item It must be a \textbf{finite} description of computation steps.
    \item Each step of the description must be as \textbf{elementary} as possible.
    \item The way the next step is achieved must be \textbf{deterministic}. 
\end{enumerate}
